# 电子商务平台

## 项目概述
这是一个完整的电子商务平台，使用 MERN (MongoDB, Express, React, Node.js) 技术栈构建。平台支持商品展示、购物车、支付系统等完整电商功能。

## 技术栈
- **后端**: Node.js, Express
- **数据库**: MongoDB
- **前端**: React, Redux
- **支付**: Stripe API
- **部署**: AWS

## 主要功能
- 用户认证和授权
- 商品分类和搜索
- 购物车管理
- 订单处理
- 支付集成
- 管理员面板

## 项目亮点
- 使用 JWT 实现安全的用户认证
- 集成 Stripe 支付系统
- 实现实时库存管理
- 响应式设计适配移动端
- 使用 Redux 管理复杂状态
